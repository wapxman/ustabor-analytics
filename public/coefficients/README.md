# Коэффициенты

Папка с данными по месяцам. Структура:

```
coefficients/
  2023/
    01.json  — Январь 2023
    02.json  — Февраль 2023
    ...
    12.json  — Декабрь 2023
  2024/
    01.json  — Январь 2024
    ...
  2025/
  2026/
```

## Формат файла

Каждый файл — JSON. Пример:

```json
{
  "month": "Январь",
  "year": 2024,
  "data": {
    "показатель_1": 123,
    "показатель_2": 456
  }
}
```

Заполните нужные поля. При обновлении сайта данные не пропадают — они хранятся в GitHub.
Чтобы обновить — просто замените файл.
