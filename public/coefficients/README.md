# Коэффициенты

Сюда загружайте Excel файлы (.xlsx) с данными по месяцам.

## Как загрузить

1. Зайдите в нужную папку года (2023, 2024, 2025, 2026)
2. Нажмите Add file → Upload files
3. Загрузите Excel файл
4. Название файла = номер месяца: `01.xlsx`, `02.xlsx`, ... `12.xlsx`

## Пример

```
coefficients/
  2024/
    01.xlsx  — Январь 2024
    02.xlsx  — Февраль 2024
    03.xlsx  — Март 2024
  2025/
    01.xlsx  — Январь 2025
```

Любой формат таблицы — сайт прочитает все колонки и строки автоматически.
Чтобы заменить — просто загрузите новый файл с тем же именем.
